## Project Overview
HandsMen Threads is a custom Salesforce CRM designed to support and streamline the operations of a menâ€™s fashion and bespoke tailoring business. The system centralizes customer management, order processing, inventory tracking, loyalty status monitoring, and marketing activities in one unified platform.

## Key Features
- Custom Salesforce objects for Customers, Orders, Products, Inventory, and Marketing Campaigns  
- Record-Triggered Flows for business process automation  
- Apex Triggers and Classes for complex logic such as order validation, inventory updates, and loyalty status management  
- Asynchronous Apex (Batch & Scheduler) for automated inventory restocking  
- Custom Lightning App and Lightning Record Pages for improved UI/UX  
- Role-based access control using Profiles and Permission Sets  

## Technologies Used
- Salesforce Lightning Experience  
- Salesforce Flows  
- Apex (Triggers, Classes, Batch, Scheduler)  
- Salesforce App Manager & Lightning App Builder  

## Project Scope
This project focuses on CRM functionality within Salesforce. External integrations such as payment gateways are excluded and intended for future enhancements.
